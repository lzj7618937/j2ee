      费用说明
      费用包含：
      1、车辆使用费
      2、全程燃油费，高速费，停车费；
      3、中文司兼导服务费；

      费用不包含：
      1、其他未标明事项，及游客个人消费；

      补充说明：
      1. 本产品需提前3天预定。
      2. 本产品为结伴拼车产品，会有其他游客一同搭乘（如果您希望自由调整，建议订购仙贝旅行的包车产品，一对一为您服务）；


      退订规则：
      1、如因客人个人原因取消服务:
      ・提前72小时取消及更改，免收退改费；
      ・提前72小时（含）至48小时取消及更改，收取订单金额的50%作为违约金；
      ・提前48小时（含）取消及更改，收取订单金额的100%作为违约金。
      2、不成团说明（成团人数为4人。若不满4人，仙贝旅行会取消行程不承担违约责任）：
      ・提前48小时取消服务，全额退款，不承担违约责任；
      ・提前48小时（含）至24小时取消服务，支付订单总额的15%作为违约金；
      ・提前24小时（含）以内取消服务，支付订单总额的30%作为违约金。
      交通方式
      基本信息：
      ・参考车型：丰田阿尔法 ・车辆座位：8座
      ・建议乘坐：6人 ・建议行李：6件
      ・服务人员：华人司兼导

      补充说明：
      1、具体车型以实际用车为主。参考车型：8座丰田阿尔法或同级。10做海狮或同级。
      2、儿童与成人价格相同，本产品暂不接受2周岁以内婴儿预定。
      3、由于结伴拼车机场接机，出发时间固定。
      4、司机会在机场等待，请您根据与司机约定时间，准时在指定地点等候。
      5、如因游客自身原因迟到，导致影响其他游客行程，司机有权拒绝等待，或不提供服务。此种情况，责任自负，概不退款。
      6、因多个航班客户共享使用本产品，如有航班延误，请提前告知仙贝，我们将会为您另行安排车辆。如无任何联络，且超时60分钟以上，将默认放弃本次服务。
      7、拼车产品，每人限带1件大行李（24寸行李箱），如因行李过多而导致司机拒载，概不承担责任。
